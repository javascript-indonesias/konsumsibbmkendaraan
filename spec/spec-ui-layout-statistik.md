# Statistik Page Layout Specification

![Status: Planned](https://img.shields.io/badge/status-Planned-blue)

Dokumen ini mendefinisikan layout untuk halaman **Statistik (statistik.html)** - menampilkan berbagai grafik dan analisis konsumsi BBM.

---

## 1. Requirements & Constraints

### Functional Requirements

- **REQ-STA-001**: Halaman HARUS menampilkan 3 jenis grafik: Line, Bar, Horizontal Bar
- **REQ-STA-002**: Setiap grafik HARUS memiliki filter rentang waktu
- **REQ-STA-003**: Grafik Line dan Bar HARUS mendukung multi-vehicle comparison
- **REQ-STA-004**: Filter kendaraan HARUS bisa toggle on/off via legend
- **REQ-STA-005**: HARUS ada opsi custom date range picker
- **REQ-STA-006**: Horizontal Bar HARUS menampilkan ranking kendaraan

### Display Constraints

- **CON-STA-001**: Grafik HARUS responsive dan readable di mobile
- **CON-STA-002**: Legend position: Bottom (mobile), Top (desktop)
- **CON-STA-003**: Tooltip HARUS format currency Rupiah dan km/L dengan benar
- **CON-STA-004**: Chart height: min 250px, max 400px

### Design Guidelines

- **GUD-STA-001**: Gunakan warna kendaraan yang konsisten di semua chart
- **GUD-STA-002**: Animasi smooth saat filter berubah
- **GUD-STA-003**: Loading skeleton saat chart rendering
