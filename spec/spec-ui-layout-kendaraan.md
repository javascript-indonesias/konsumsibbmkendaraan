# Kendaraan Page Layout Specification

![Status: Planned](https://img.shields.io/badge/status-Planned-blue)

Dokumen ini mendefinisikan layout untuk halaman **Kelola Kendaraan (kendaraan.html)** - CRUD kendaraan.

---

## 1. Requirements & Constraints

### Functional Requirements

- **REQ-VEH-001**: Halaman HARUS menampilkan daftar semua kendaraan
- **REQ-VEH-002**: HARUS ada tombol untuk menambah kendaraan baru
- **REQ-VEH-003**: Setiap kendaraan HARUS punya action: Edit, Hapus
- **REQ-VEH-004**: Form kendaraan HARUS di modal (tidak navigate ke page baru)
- **REQ-VEH-005**: HARUS menampilkan jumlah pengisian per kendaraan
- **REQ-VEH-006**: Warna kendaraan HARUS bisa dipilih dari palette
- **REQ-VEH-007**: HARUS ada konfirmasi sebelum hapus kendaraan

### Validation Rules

- **VAL-VEH-001**: Nama kendaraan: 1-50 karakter (required)
- **VAL-VEH-002**: Jenis: "motor" atau "mobil" (required)
- **VAL-VEH-003**: Tahun: 1900-current year+1 (opsional)
- **VAL-VEH-004**: Plat nomor: maks 15 karakter (opsional)

### Design Guidelines

- **GUD-VEH-001**: Card-based list untuk kendaraan
- **GUD-VEH-002**: Visual indicator warna kendaraan
- **GUD-VEH-003**: Modal slide dari bawah di mobile
